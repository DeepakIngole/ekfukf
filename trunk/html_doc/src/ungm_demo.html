DESCRIPTION 

 Demonstration of univariate nonstationary growth model (UNGM) using
 EKF and UKF.

CROSS-REFERENCE INFORMATION

This function calls:
  • ungm_df_dx Jacobian of the state transition function for the UNGM-model.
  • ungm_dh_dx Jacobian of the measurement model function for the UNGM-model.
  • ungm_f State transition function for the UNGM-model.
  • ungm_h Measurement model function for the UNGM-model.
  • ekf_predict1 EKF_PREDICT 1st order Extended Kalman Filter prediction step
  • ekf_update1 EKF_UPDATE 1st order Extended Kalman Filter update step
  • erts_smooth1 ERTS_SMOOTH1 Extended Rauch-Tung-Striebel smoother
  • gauss_pdf GAUSS_PDF Multivariate Gaussian PDF
  • gauss_rnd GAUSS_RND Multivariate Gaussian random variables
  • resampstr RESAMPSTR Stratified resampling
  • ukf_predict3 UKF_PREDICT2 Augmented form Unscented Kalman Filter prediction step
  • ukf_update3 UKF_UPDATE2 - Augmented form Unscented Kalman Filter update step
  • urts_smooth1 URTS_SMOOTH1 Additive form Unscented Rauch-Tung-Striebel smoother
