bot_dh_dx

PURPOSE

Jacobian of the measurement model function in BOT demo.

SYNOPSIS

function dY = bot_dh_dx(x,s)

DESCRIPTION 

 Jacobian of the measurement model function in BOT demo.

  dh_dx = -(y-sy) / (x-sx)^2 * 1 / (1 + (y-sy)^2 / (x-sx)^2)
        = -(y-sy) / ((x-sx)^2 + (y-sy)^2)
  dh_dy = 1 / (x-sx) * 1 / (1 + (y-sy)^2 / (x-sx)^2)
        = (x-sx) / ((x-sx)^2 + (y-sy)^2)
