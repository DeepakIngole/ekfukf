az_dh_dx

PURPOSE

AZ_DH_DX Measurement derivative function for EKF.

SYNOPSIS

function dY = az_dh_dx(x,s)

DESCRIPTION 

AZ_DH_DX Measurement derivative function for EKF.

  dh_dx = -(y-sy) / (x-sx)^2 * 1 / (1 + (y-sy)^2 / (x-sx)^2)
        = -(y-sy) / ((x-sx)^2 + (y-sy)^2)
  dh_dy = 1 / (x-sx) * 1 / (1 + (y-sy)^2 / (x-sx)^2)
        = (x-sx) / ((x-sx)^2 + (y-sy)^2)
